# 5.2 Build Guide

## Dependencies

```bash
sudo apt install libopencv-dev libyaml-cpp-dev
```

## Build

Clone the repository:

```bash
git clone https://github.com/M11158002/FisheyeTransformer.git
cd FisheyeTransformer
```

Create a build directory:

```bash
mkdir build
cd build
```

Configure the project:

```bash
cmake ..
```

Build the project:

```bash
make
```
